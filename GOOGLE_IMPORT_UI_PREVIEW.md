## Design Elements

### Import Button
- **Background**: White with gradient border (Google colors)
- **Icon**: Official Google logo (4 colors)
- **Text**: Bold, clear call-to-action
- **Hover**: Subtle scale and shadow effect
- **Tap**: Scale down animation

### Modal Design
- **Header**: Gradient background with Google brand colors
- **Cards**: Rounded corners with subtle shadows
- **Icons**: Emoji and Lucide icons for clarity
- **Colors**: 
  - Success: Green (#10B981)
  - Info: Blue (#3B82F6)
  - Error: Red (#EF4444)
  - Google: Multi-color gradient

### Interaction States
- **Default**: Clean, inviting
- **Hover**: Subtle highlight
- **Active**: Pressed state
- **Loading**: Spinning loader
- **Success**: Checkmark animation
- **Error**: Alert icon with details

## Mobile Responsiveness

### Phone View (< 640px)
- Full-width import button
- Modal takes 90% of screen
- Large touch targets
- Readable text sizes
- Optimized for one-hand use

### Tablet View (640px - 1024px)
- Centered content
- Max-width constraints
- Comfortable spacing
- Balanced layout

### Desktop View (> 1024px)
- Compact modal (max 500px wide)
- Centered on screen
- Hover states active
- Drag & drop emphasized
